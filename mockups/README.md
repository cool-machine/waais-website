# WAAIS Mockups

Design artifacts for review before Vue/Laravel implementation.

- `design-system.html` — visual system, tokens, components, and public/app surface rules.
- `public-site.html` — clickable public website prototype for Home, Events, Startups, About, Partners, Get Involved, Contact, and Legal.
- `app-dashboard-admin-auth.html` — interactive auth, member dashboard, and admin dashboard prototype.
- `assets/waais-hero-video.mp4` — local homepage hero video downloaded from George's licensed YouTube upload.

Production code should not live here. The planned implementation folders are:

- `/frontend/` for the Vue 3 app.
- `/backend/` for the Laravel app.
